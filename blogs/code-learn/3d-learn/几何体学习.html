    # 模型名称

    💬为什么要给模型命名？

    一般实际开发的时候，会加载外部的模型，然后从模型对象通过节点的名称.name查找某个子对象。使用的方法是父对象.getObjectByName (模型名)。

        	const group = new THREE.Object3D();
        group.add(mesh);
        group.add(mesh2);
        console.log(`查找相应的模型->`,group.getObjectByName('test1'))

		// Group 继承基类Object3D的方法
		const group = new THREE.Group();
        group.add(mesh);
        group.add(mesh2);
        console.log(`查找相应的模型->`,group.getObjectByName('test1'))

    # 遍历生成多个模型

    案例：生成小区的楼栋模型

    	 	const scene = new THREE.Scene();

        const group1 = new THREE.Group();
        group1.name = '高层'
        for (let i = 0; i < 5; i++) {
            const geometry = new THREE.BoxGeometry(20, 60, 10);
            const material = new THREE.MeshStandardMaterial({ color: 0xf6d572 });
            const mesh = new THREE.Mesh(geometry, material);
            mesh.position.set(i * 30, 0, 0);//沿着x轴排布
            mesh.name = `A${i + 1}号楼`
            group1.add(mesh);
        }
        group1.position.set(0, 30, 0);

        const group2 = new THREE.Group();
        group2.name = '居民楼'
        for (let i = 0; i < 5; i++) {
            const geometry = new THREE.BoxGeometry(20, 30, 10);
            const material = new THREE.MeshPhysicalMaterial({ color: 0xf6d572 })
            const mesh = new THREE.Mesh(geometry, material);
            mesh.position.set(i * 30, 0, 0);//沿着x轴排布
            mesh.name = `B${i + 1}号楼`
            group2.add(mesh);
        }
        group2.position.z = 50;
        group2.position.y = 15;
        const model = new THREE.Group();
        model.name = '小区'
        model.add(group1, group2)
        model.position.set(-50, 0, -25);
        console.log(`model->`, model)

        // 遍历模型设置颜色
        model.traverse((obj) => { // traverse: 遍历模型中的所有节点
            if (obj.isMesh) {
                obj.material.color.set(0xffff00)
            }
        })
        // 给某栋楼单独的楼设置颜色
        const obj = group1.getObjectByName('A3号楼');
        obj.material.color.set(0xff0000)
        scene.add(model);

        // 环境光
        const ambientLight = new THREE.AmbientLight(0xffffff, 0.4);
        scene.add(ambientLight);

        // 平行光
        const directionalLight = new THREE.DirectionalLight(0xffffff, 1);
        directionalLight.position.set(0, 0, 100);
        directionalLight.target = model
        scene.add(directionalLight);

		//相机
        const camera = new THREE.PerspectiveCamera(45, window.innerWidth / window.innerHeight, 1, 1000);
        camera.position.set(0, 0, 100);
        camera.lookAt(model.position);

		//渲染器
        const renderer = new THREE.WebGLRenderer();
        renderer.setSize(window.innerWidth, window.innerHeight);
        renderer.render(scene, camera)
        document.body.appendChild(renderer.domElement);

        // 模型可交互
        const controls = new OrbitControls(camera, renderer.domElement)
        controls.addEventListener('change', () => {// 如果OrbitControls改变了相机参数，要重新渲染
            renderer.render(scene, camera)
        })
        // 坐标辅助查看
        const axesHelper = new THREE.AxesHelper(50);
        scene.add(axesHelper);

    # 纹理材质（纹理贴图）

    什么是纹理材质呢？想不想diy模型的外表? 方法就是在模型的外表上加载自己的图片!

    • 纹理加载器：TextureLoader()
    • 实例化纹理加载器得到纹理对象，调用**load()**方法加载图片

    🔔在材质中同时设置纹理贴图map和颜色color，会颜色混合（真的超级惊艳！意外的好看呐！）

            // 平面缓冲几何体:用于生成平面几何体
        const PlaneGeometry = new THREE.PlaneGeometry(100, 100);
        // 纹理加载器
        const textLoader = new THREE.TextureLoader();
		// 使用load()方法记载图片，得到一个纹理对象
        const texture = textLoader.load('./imgs/heart.png');

		const geometry = new THREE.BoxGeometry(20, 60, 10);
        const material = new THREE.MeshStandardMaterial({
              color: 0xf6d572,
              map: texture//设置材质的颜色贴图属性
            });
